Tematem projektu numer 2 jest platforma pełniąca rolę wirtualnego dziekanatu, umożliwiająca przeglądnięcie podstawowych danych o studentach (imię, nazwisko, data urodzenia, numer PESEL, imiona rodziców, oceny z poszczególnych przedmiotów jak również średnią). Serwis utworzono w oparciu o HTML, CSS oraz Javascript obsługiwany po stronie użytkownika.
